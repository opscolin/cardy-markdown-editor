# Cardy Markdown Editor 功能说明文档

本文档详细说明了 Cardy Markdown Editor 当前支持的所有 Markdown 语法及其展现效果。

## 1. 基础语法 (Basic Syntax)

### 标题 (Headings)
支持 H1 到 H4 四级标题，字号随全局设置等比缩放。
- `# 标题一`
- `## 标题二`
- `### 标题三`
- `#### 标题四`

### 文本样式
- **加粗**: `**文本**`
- *斜体*: `*文本*`
- ~~删除线~~ (GFM): `~~文本~~`
- **引用块**: `> 引用内容` (带垂直装饰线与斜体效果)

### 列表 (Lists)
- **无序列表**: `*` 或 `-` 开头。
- **有序列表**: `1.` 开头。
- **任务列表** (GFM):
  - [ ] 未完成任务
  - [x] 已完成任务

## 2. 高阶功能 (Advanced Features)

### 表格 (Tables - GFM)
支持标准 GFM 表格语法，包含对齐方式。针对卡片展示进行了响应式滚动优化。
```markdown
| 项目 | 价格 | 备注 |
| :--- | :--: | ---: |
| 基础版 | $10 | 个人使用 |
| 专业版 | $30 | 团队协作 |
```

### 图片处理 (Images)
- 支持从本地上传或直接粘贴。
- **持久化**: 图片自动转换为 Base64 编码并存储在 Markdown 中，确保跨设备/会话可用。
- **样式**: 自动生成圆角、阴影及边框，居中展示。

### 代码块 (Code Blocks)
- **行内代码**: `` `code` `` (橘色背景高亮)。
- **块级代码**: 
  - 支持多行代码展示。
  - **样式**: 内置 Mac 风格装饰（红、黄、绿三色圆点柜）。
  - **自适应**: 自动处理长文本换行，防止溢出。

### 分页符 (Divider/Cards)
- 使用 `---` 创建新的卡片。Cards 布局会自动识别此符号并进行分页处理。

## 3. 演示模式 (Presentation Mode)
- **垂直幻灯片**: 点击顶部 “Present” 按钮进入全屏演示模式。
- **导航控制**:
  - **键盘**: 支持左右/上下方向键以及空格键切换。
  - **手势**: 支持鼠标拖拽或手机滑动手势进行翻页。
  - **自动缩放**: 幻灯片根据屏幕尺寸自动缩放，确保在任何设备上完美展现。

## 4. 导出功能 (Export Features)
- **PDF 导出**: 将所有卡片导出为高质量的 PDF 文件。
- **ZIP 导出**: 将所有卡片分别生成图片并打包为 ZIP 压缩包。
- **Markdown 导出**: 导出当前笔记的原始 Markdown 源码。
- **单页导出**: 在预览区悬浮在特定卡片上，点击下载图标可单独导出该页为 PNG。

## 5. 全局设置 (Global Settings)

- **字号等比缩放**: 提供 XS / SM / BASE / LG 四档，标题与正文按比例整体变动。
- **背景网格**: 可开启点阵背景。
- **自定义页脚**: 支持为每张卡片底部添加唯一的署名或备注。
